# Portas de Conexão – WALL-E e EVA

## WALL-E

###  Motores
| Motor | Posição          |
|--------|------------------|
| M3     | Frente Direita   |
| M1     | Frente Esquerda  |
| M2     | Atrás Direita    |
| M4     | Atrás Esquerda   |

###  Ultrassônico
| Função  | Porta |
|----------|-------|
| Trigger  | A0    |
| Echo     | A1    |

###  Infravermelho
| Posição | Porta |
|----------|-------|
| Frente   | A4    |
| Atrás    | A5    |


---

##  EVA

###  Motores
| Motor | Lado     |
|--------|-----------|
| M2     | Direita   |
| M1     | Esquerda  |

###  Infravermelho
| Tipo   | Porta |
|--------|-------|
| Único  | A5    |

###  Ultrassônico
| Função  | Porta |
|----------|-------|
| Trigger  | A0    |
| Echo     | A1    |

---

**Observação:**  
As portas estão nomeadas conforme o padrão de conexão utilizado na placa Arduino
